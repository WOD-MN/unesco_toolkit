For Heritage Conservation Across the Globe!

Digitization Best Practices

Oral History Documentation

  • Record in lossless formats (WAV, FLAC) at minimum 44.1kHz sample rate
  • Include detailed contextual metadata about interviewee and context
  • Store master files separately from access copies
  • Create transcripts with timecodes for important sections

3D Object Scanning

  • Use appropriate resolution based on object size and detail
  • Capture from multiple angles with sufficient overlap
  • Include scale reference in scans
  • Document lighting conditions and equipment used

Metadata Standards

Recommended Standards

  • Use Dublin Core for basic descriptive metadata
  • Implement METS for complex digital objects
  • Follow PREMIS for preservation metadata
  • Adopt CIDOC CRM for cultural heritage documentation

File Naming Conventions

  • Use consistent, meaningful naming patterns
  • Include date in YYYYMMDD format when relevant
  • Avoid special characters and spaces
  • Implement version control in filenames

Community Engagement

Participatory Documentation

  • Involve community members in the documentation process
  • Respect indigenous knowledge protocols
  • Provide training on documentation techniques
  • Establish clear agreements about access and use

Ethical Considerations

  • Obtain informed consent for documentation
  • Respect cultural sensitivities and restrictions
  • Establish protocols for culturally sensitive materials
  • Provide mechanisms for ongoing community control

Digital Preservation

File Formats

  • Use open, standardized formats when possible
  • Maintain master files in lossless formats
  • Create access copies in common formats
  • Document all format decisions

Storage and Backup

  • Implement the 3-2-1 backup rule (3 copies, 2 media types, 1 offsite)
  • Use checksums for file integrity verification
  • Plan for regular format migration
  • Document all preservation actions
